# BMI Calculator

A simple and responsive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript.

## Features

* Calculate BMI instantly
* Responsive design for mobile and desktop
* Clean and modern UI
* BMI category message display
* Input validation for height and weight

## Technologies Used

* HTML5
* CSS3
* JavaScript

## BMI Categories

* Under Weight = Less than 18.6
* Normal Range = 18.6 to 24.9
* Overweight = Greater than 24.9

## How to Use

1. Enter your height in centimeters
2. Enter your weight in kilograms
3. Click on the **Calculate** button
4. Your BMI result and category will be displayed
